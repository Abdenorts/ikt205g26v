# Fast Notes(React Native / Expo)

This is a minimal application to store your notes. 
The notes will only be stored in memory (No persistent storage, as of yet).
This application is written for Android only.

To launch the application you will need:
- Node.js
- npm
- Expo CLI
- Android Studio for android emulator, or a personal mobile device
- Expo Go (only a requirement for phisical mobil phones)

To run the application, go to your expo CLI.

Type: 'npx expo start'

This will give you some options, if you want to open the applicattion in the emulator, Type a.

If you are on a physical mobile device, go to your Expo Go app and scan the QR code from expo CLI.

This will give you access to the application. 


Git repository: https://github.com/Abdenorts/ikt205g26v/tree/main/react-native/FastNotes
